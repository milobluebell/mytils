> **mytils** 是前端开发中，常用到的一些工具函数和算法的集合。

比如 **时间格式的转换、倒计时展示、省略小数点后 N 位、笛卡尔乘积** 以及更多...

---

### 安装

##### npm

```shell
npm install mytils --save
```

##### yarn

```shell
yarn add mytils --save
```
